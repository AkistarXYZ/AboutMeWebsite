# AboutMe Website

This repository contains the source code for my personal website.

The website was built using plain HTML, CSS, and JavaScript as a way to practice web development while creating a place to share my projects.

## Features

* About Me page
* Games page
* Links to my YouTube channel
* Animated card carousel
* Interactive UI elements
* Hidden easter eggs

## Built With

* HTML
* CSS
* JavaScript

## Purpose

The goal of this project was to improve my front-end development skills while creating a website that I can continue expanding as I make more games and projects.

## Future Plans

* Add more games
* Improve the design
* Add more animations and interactions
* Continue updating the site as new projects are released
